# List Built-in Tools

List all core, built-in non-MCP development tools available to you.

## Format requirements:
- Group tools by category (File Operations, Search, Execution, Task Management, Web, Planning)
- Use TypeScript function syntax with parameters
- Mark required parameters, optional parameters should have `?` suffix
- Include a brief one-line description for each tool

## Example output format:
```
### File Operations
- `Read(file_path: string, offset?: number, limit?: number)` - Read file contents with optional line range
```
